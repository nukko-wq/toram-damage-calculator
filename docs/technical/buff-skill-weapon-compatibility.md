# バフスキル武器種対応表

## 概要
トーラムオンラインのバフスキルは、メイン武器とサブ武器の組み合わせによって使用可能なスキルが変わります。
この文書では各武器構成で使用可能なバフスキルを整理します。

## メイン武器：片手剣 の場合

- **サブ武器なし**
- **サブ武器：ナイフ**
- **サブ武器：盾**
- **サブ武器：手甲**

### 特殊パターン（サブ武器：矢）

**スキル数：合計59スキル（基本58 + 追加1）**

## スキル分類統計

### レベル設定あり（基本パターン）
- **1-10レベル**：26スキル
- **1-3レベル**：1スキル (神速の捌手)
- **特殊レベル**：2スキル (ブレイブオーラ-レベル2相当、熱情の歌-重ねがけ×10)

### ON/OFF切り替えのみ（基本パターン）
- **29スキル**

### 矢装備時の追加
- **ON/OFF切り替え**：1スキル (パライズショット)

### 魔道具装備時の追加
- **レベル設定**：2スキル (急速チャージ-レベル1-15、レゾナンス？-レベル1-10)
- **ON/OFF切り替え**：3スキル (P:コンバージョン、エンチャントバースト、デュアルブリンガー-旧)

### 巻物装備時の追加
- **ON/OFF切り替え**：1スキル (風遁の術)

|-----------|----------|-----------------|------|

| 片手剣 | 矢 | 59 | パライズショット追加 |
| 片手剣 | 巻物 | 59 | 風遁の術追加 |
| 片手剣 | 魔道具 | 63 | マジック系スキル5個追加 |
