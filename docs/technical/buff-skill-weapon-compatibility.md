# バフスキル武器種対応表

## 概要
トーラムオンラインのバフスキルは、メイン武器とサブ武器の組み合わせによって使用可能なスキルが変わります。
この文書では各武器構成で使用可能なバフスキルを効率的に整理します。

## 共通バフスキル（全武器種対応）

すべての武器種で使用可能な基本バフスキル群です。**35スキル**

### ブレードスキル
- **ウォークライ** (IsWarcry) - ON/OFF
